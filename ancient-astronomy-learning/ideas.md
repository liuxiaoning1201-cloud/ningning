# 古代天文文化學習網站 - 設計規劃

## 內容分析

PPT 包含 30 張投影片，主要涵蓋：
1. **七曜與五星** - 日月金木水火土的古代命名與文化意義
2. **四象與四靈** - 青龍、白虎、玄武、朱雀的方位對應
3. **二十八星宿** - 黃道分區與四象的關係
4. **北斗七星** - 季節指示與天極
5. **黃道與天體運行** - 古代天文觀測基礎
6. **三垣** - 天文分區系統
7. **實踐課題** - 學生動手製作星宿盒子

## 選定設計方案

**設計哲學：古韻現代 (Classical Astronomy Meets Modern Learning)**

### 核心原則
1. **時空對話** - 將古代天文知識與現代學習體驗結合，營造「穿越時空的學習」氛圍
2. **視覺層次** - 利用深色背景模擬夜空，星辰元素貫穿全站
3. **互動探索** - 每個知識點都可展開探索，鼓勵學生主動學習
4. **文化尊重** - 保留中文古籍引文與傳統美學，不過度現代化

### 色彩哲學
- **主色調**：深藍/深紫（夜空）+ 金色/銀色（星辰）
- **背景**：深色系（#0f172a 或 #1a1a2e），營造夜空感
- **強調色**：金色（#d4af37）用於重點、標題、互動元素
- **文字**：淺色（#f0f0f0、#e0e0e0），確保可讀性
- **情感**：神秘、優雅、知識性、啟蒙感

### 佈局範式
- **非對稱設計**：避免單調的居中佈局
- **卡片式內容**：每個知識點獨立卡片，可展開/摺疊
- **星圖視覺**：首頁使用星圖背景，營造沉浸感
- **側邊導航**：固定導航欄，快速切換章節

### 簽名元素
1. **星辰點綴** - 頁面各處散佈小星星動畫
2. **古籍引文框** - 特殊樣式呈現古代文獻
3. **星宿圖示** - 自製或生成的二十八宿圖標

### 互動哲學
- **漸進式展開** - 點擊卡片展開詳細內容
- **星辰懸停效果** - 滑鼠懸停時星星發光
- **平滑過渡** - 所有動畫使用 Framer Motion 實現流暢效果
- **進度追蹤** - 顯示學習進度條

### 動畫指南
- **入場動畫**：卡片從下方淡入滑上（stagger effect）
- **懸停效果**：卡片輕微上浮、陰影加深、邊框發光
- **展開動畫**：內容區域平滑展開，高度漸變
- **星星動畫**：細微的閃爍與浮動效果
- **頁面轉換**：淡入淡出過渡

### 排版系統
- **標題字體**：思源宋體（Noto Serif SC）- 古典、優雅
- **正文字體**：思源黑體（Noto Sans SC）- 清晰、易讀
- **標題層級**：
  - H1：36px，金色，思源宋體，用於頁面標題
  - H2：28px，銀色，思源宋體，用於章節標題
  - H3：20px，淺色，思源黑體，用於小節標題
  - 正文：16px，#e0e0e0，思源黑體

## 技術實現

- **框架**：React 19 + Tailwind CSS 4
- **動畫**：Framer Motion
- **UI 組件**：shadcn/ui
- **字體**：Google Fonts (思源宋體、思源黑體)
- **圖片資源**：生成式 AI 創建星圖背景、古籍紋理
- **互動**：展開/摺疊、標籤頁、進度追蹤

## 設計優先級

1. **第一階段**：首頁 + 七曜章節 (基礎框架驗證)
2. **第二階段**：四象 + 二十八星宿 (核心內容)
3. **第三階段**：北斗 + 天文基礎 (補充知識)
4. **第四階段**：課堂任務 + 互動功能 (學習支援)
