# 班級守護隊 — 專案文件總覽

本文件列出專案內所有重要檔案，方便在左側檔案欄快速對照與開啟。

---

## 根目錄

| 檔案 | 說明 |
|------|------|
| `index.html` | 開發用 HTML 入口，載入 `/src/main.ts` |
| `package.json` | 依賴與腳本（dev / build） |
| `package-lock.json` | 鎖定依賴版本 |
| `vite.config.ts` | Vite 設定（Vue、Tailwind、base path） |
| `tsconfig.json` | TypeScript 根設定 |
| `tsconfig.app.json` | 應用程式 TS 設定 |
| `tsconfig.node.json` | Node/Vite 環境 TS 設定 |

---

## src/ — 主要程式碼

| 檔案 | 說明 |
|------|------|
| `main.ts` | 應用程式入口，掛載 Vue、Pinia、Router |
| `App.vue` | 根元件，左側導覽列與 `<router-view>` |
| `style.css` | 全域樣式與 Tailwind、動畫 |

---

## src/types/

| 檔案 | 說明 |
|------|------|
| `index.ts` | 型別定義、常數：Student / Group / ScoreEvent / Badge、ANIMAL_CONFIG、DEFAULT_SCORE_BUTTONS、ENCOURAGEMENTS_BY_CATEGORY、CATEGORY_LABELS |

---

## src/stores/

| 檔案 | 說明 |
|------|------|
| `classStore.ts` | Pinia store：班級／學生／小組／評分事件／徽章、localStorage 持久化、加減分與撤銷 |

---

## src/router/

| 檔案 | 說明 |
|------|------|
| `index.ts` | Vue Router 路由：總覽、課堂記分、學生展示、歷史、設定（Hash 模式） |

---

## src/views/ — 頁面

| 檔案 | 說明 |
|------|------|
| `TeacherDashboard.vue` | 總覽：小組卡片、今日統計、即時動態、快速前往 |
| `ClassroomControl.vue` | 課堂記分：小組／學生選擇、8 個計分按鈕、隨機鼓勵語 Toast（約 5 秒）、撤銷、今日動態 |
| `StudentDisplay.vue` | 學生展示（投影）：三組進度、星星與進度條、鼓勵語輪播、無精確分數 |
| `HistoryView.vue` | 歷史紀錄：時間／小組／類別篩選、摘要、類別分佈、事件時間軸、週報與成長亮點 |
| `SettingsView.vue` | 設定：班級資訊、小組名稱、學生新增／編輯／刪除、批量貼上、清除今日／重置全部 |

---

## src/components/ — 共用元件

| 檔案 | 說明 |
|------|------|
| `AnimalMascot.vue` | 動物吉祥物（貓頭鷹／海獺／柴犬），可選尺寸與動畫 |
| `GroupCard.vue` | 小組卡片：吉祥物、組名、今日進度條、成員頭像、徽章 |
| `ScoreButton.vue` | 計分按鈕：emoji、標籤、分數、依類別配色 |
| `StudentChip.vue` | 學生選擇膠囊：座號、姓名、選中時小組色高亮 |
| `BadgeDisplay.vue` | 徽章展示（解鎖的徽章列表） |

---

## 其他

| 路徑 | 說明 |
|------|------|
| `.vscode/extensions.json` | 建議的 VS Code 擴充套件 |
| `node_modules/` | 依賴（勿手動改） |
| `dist/` | 建置輸出（執行 `npm run build` 產生） |

---

## 常用指令

- 本機開發：`npm run dev`
- 建置：`npm run build`
- 字遊空間部署（在專案外層執行）：`npm run pages:deploy`

---

## 資料儲存

- 所有班級／學生／評分／徽章資料存於 **瀏覽器 localStorage**，鍵名：`class-guardian-data`。未上傳伺服器，換裝置或清除網站資料會不見。
