# 🎁 手勢點名系統

一款採用可愛粉色調與圓潤 UI 設計的互動式點名系統，整合了 **MediaPipe Hands** 進行即時手勢識別。

## ✨ 功能亮點

1. **可愛視覺設計**：使用 CSS 繪製了一個會跳動的可愛禮盒，並帶有表情動畫
2. **手勢控制**：當教師對著鏡頭**「張開一隻手」**時，禮盒會自動彈開
3. **名單管理**：在設置頁面輸入學生名單，系統會自動隨機抽取
4. **特效**：禮盒開啟時會有五彩紙屑（Confetti）特效，增加課堂驚喜感

## 🎯 設計說明

### 1. 可愛外觀
- 使用 `#ff6b9d`、`#ff8fab` 等柔和的粉色系色彩
- 禮盒在待機狀態下會持續跳動（`jump` 動畫），吸引學生注意力
- 禮盒帶有可愛的笑臉表情，並有眨眼動畫

### 2. MediaPipe Hands 識別邏輯
- **手勢識別**：只識別手勢，不識別面部
- **張開手判斷**：當檢測到至少 3 個手指張開時，判定為手掌張開
- **座標歸一化**：MediaPipe 提供的是 0 到 1 之間的座標，使得識別在不同屏幕尺寸下都能正常運作

### 3. 防誤觸設計
- **冷卻時間**：設置了 `TRIGGER_COOLDOWN` 為 3 秒，避免一次動作連續觸發多次點名
- **自動重置**：禮盒開啟後，姓名會懸浮 5 秒，隨後自動關閉並回到待機狀態

### 4. 名單儲存
- 使用 `localStorage` 儲存名單
- 教師在設置頁面輸入名單後，即使刷新頁面或關閉瀏覽器，名單依然會被保存

## 🚀 如何使用

1. 打開 `index.html` 文件（建議使用現代瀏覽器，如 Chrome、Edge、Firefox）
2. 點擊「名單設置」，輸入您的班級名單（每行一個名字）
3. 點擊「儲存名單」保存名單
4. 點擊「返回」回到主頁面
5. 點擊「開始點名」，允許瀏覽器訪問您的攝像頭
6. 當您想抽人的時候，對著鏡頭**優雅地張開手掌**，看是哪位幸運的小可愛被選中！
