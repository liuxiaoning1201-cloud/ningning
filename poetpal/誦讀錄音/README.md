# 建議篇章誦讀錄音（第一至第四學習階段）

本資料夾存放香港教育局四個學習階段建議篇章的**誦讀錄音**，已統一為 **WAV** 格式（原為 MP3，已轉檔以改善相容性與播放穩定性），供詩友記（PoetPal）專案使用。

## 資料夾結構

| 資料夾 | 學習階段 | 篇章數 | 錄音說明 |
|--------|----------|--------|----------|
| **KS1** | 第一學習階段（小一至小三） | 20 篇 | 粵語（c）、普通話（p），部分篇章僅有普通話 |
| **KS2** | 第二學習階段（小四至小六） | 20 篇 | 粵語（c）、普通話（p） |
| **KS3** | 第三學習階段（中一至中三） | 25 篇 | 粵語台詞誦（c）、粵語吟誦（y）、普通話（p），部分為 c1/c2 或 a/b 子篇 |
| **KS4** | 第四學習階段（中四至中六） | 20 項（24 篇） | 粵語台詞誦（c/c1/c2）、粵語吟誦（y）、普通話（p） |

## 檔名說明

- **賞析 PDF**：`KSx_NN_篇名.pdf`（例：`KS1_01_守株待兔.pdf`、`KS4_16a_山居秋暝.pdf`）
- **誦讀錄音**：格式為 **WAV**，檔名為 `KSx_NN_篇名(粵語).wav` / `篇名(普通話).wav` / `篇名(粵語吟誦).wav`，同一篇有多段粵語時為 `(粵語2)` 等
- **c**：粵語台詞誦 → 檔名備註 **(粵語)**
- **y**：粵語吟誦 → 檔名備註 **(粵語吟誦)**
- **p**：普通話 → 檔名備註 **(普通話)**
- **c1 / c2**：同一篇多段粵語台詞誦（如 KS4 念奴嬌）→ **(粵語)**、**(粵語2)**
- **a / b**：同一編號下之子篇（如古詩兩首、世說新語兩則）

範例：`KS4_16b_月下獨酌其一(粵語吟誦).wav` = 第四階段第 16 項第 b 篇（月下獨酌）之粵語吟誦。
